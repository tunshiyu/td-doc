## ChartPlot

是图表组件的生成 DOM，getDom 属性接受各种生成图表的方法。

### 接受参数

| 属性   | 说明                                                                                    | 类型                                                                                 | 必填 | 默认值 |
| ------ | --------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ | ---- | ------ |
| getDom | 接受生成图表的方法和配置，返回 Layer 对象可用于[状态管理](../../../charts/senior-usage) | (dom: HTMLElement) => create 图表方法([配置说明](../../../charts/utils/base-config)) | true |        |
